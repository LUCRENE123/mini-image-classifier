# Mini Image Classifier (MNIST Digits)

## Projet
Projet de versioning Git avec expérimentation de modèles ML sur MNIST.

## Objectif
- Utilisation de Git (branches, merge, conflit)
- Comparaison de modèles ML
- Pipeline de classification d'images

## Dataset
MNIST digits (sklearn)

## Data Scientist A
- Random Forest Classifier

## Data Scientist B
- Logistic Regression
- Normalisation des données (StandardScaler)

## Workflow Git
- feature-model-alpha (A)
- feature-model-beta (B)
- merge conflict
- résolution
- tag final

## Résultat
Comparaison des performances des deux modèles.
